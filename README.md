## Hospital Management System

### 📝 Description
A Python-based Hospital Management System. This project implements a complete medical records system with user authentication, role-based access control, and JSON file storage.

### 🚀 Features
- 🔐 **User Management & Authentication** - Secure login and password recovery
- 👥 **Role-Based Access Control** - Varying levels of access based on role (admin, doctor, nurse, patient)
- 📋 **Patient Record Management** - Personal details, medical history, doctor's notes, appointment history
- 📅 **Appointment & Scheduling System** - Online booking, rescheduling, status tracking
- 🔧 **Record Maintenance (Admin Only)** - Modules for user accounts and patients information

### 🛠️ Technologies
- Python
- JSON

- Data modeling for healthcare systems
